整理一些特殊场景的第三方库
2023-09-26 21:05:03 阅读量 #工作

整理一些工作中遇到的特殊场景会用到的开源库，大部分都比较基础，可能一辈子都用不上，但是还是得心里有个数。

虚拟滚动

Virtuoso

虚拟滚动的场景在工作中并不常见，就算见到也不好直接手写，用开源的库会好一些，通常情况下，可以封装成一个通用的基础组件。

React Hook

aHooks

开源好用的hook库，这个还是比较常用的。

H5表单

rc-field-form

封装基础Form组件会用到

WEB 3D

Babylon

基于webgl的3d渲染引擎，值得学习。

移动端拖拽&动画

@use-gesture/react

@react-spring/web

这俩兄弟相辅相成，可以做很多移动端的拖拽动画效果，比如下拉刷新。

文件操作相关

find-up 寻找当前目录下的文件

fs-extra 文件读写

comment-json 更好的处理json文件

path 路径处理

prettier 格式化文件

高亮

driver.js

h5预览高亮dom会用到，或者一些新手引导会用到
